# Smart Interview Tracker

This is a full-stack application built to track job applications, interview stages, and progress in an organized way.

The goal of this project was to build a structured dashboard-style application connected to real backend APIs.

## 🚀 Tech Stack

- React
- Node.js
- Express
- MongoDB
- JWT Authentication

## ✨ Features

- Add job applications
- Track interview stages
- Update status
- Delete entries
- Secure authentication

## 📁 Project Structure

/backend  
Express server & MongoDB models  

/frontend  
React application
